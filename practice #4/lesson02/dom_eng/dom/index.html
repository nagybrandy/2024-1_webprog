      Task 9: Counter Component

      Create a counter component!
      a. The counter consists of a read-only text input field and two buttons (plus and minus)! Clicking the buttons should increase or decrease the number in the text input field by one.
      b. Define a minimum and maximum value in the script! If the counter reaches either value, the corresponding button should be disabled!
      c. *If the button is held down, the counter should automatically start changing the value.

      Task 10: Guess the Number

      The computer is thinking of a number! Our task is to guess it. Provide the ability to guess the number, and the computer will respond whether its number is smaller or larger than our guess. Use the Math.random() function for generating a random number and Math.floor() for rounding!
